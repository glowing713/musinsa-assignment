## 무신사 프론트엔드 사전과제

[고민 과정 / 부연 설명](https://circular-kitchen-50d.notion.site/b3ae557d65d84fa99ae67b4017048ec4)

- Clone Project
```
git clone https://github.com/glowing713/musinsa-assignment.git
cd musinsa-assignment
yarn install
```

- Start react application
```
yarn start
```

- Dependencies
